================================================================================
                Pancake: Developer Command Line Tool v1.0.0
================================================================================

Thank you for installing Pancake! This document provides important 
information about Pancake, including its features, installation instructions, 
troubleshooting tips, and contact details.

---------------------------------------------------------------------------------
1. DESCRIPTION
---------------------------------------------------------------------------------

Pancake streamlines your project management workflow, simplifies syncing 
multiple projects from remote repositories, helps build and run projects 
on a local machine, and allows for the installation, uninstallation, and 
updating of development software and tools. No need to remember longer commands. 
Share your entire development setup with your work buddies or back up 
your entire development setup by sharing `$HOME/pancake.yml`.

---------------------------------------------------------------------------------
2. SYSTEM REQUIREMENTS
---------------------------------------------------------------------------------

No specific requirements. Pancake runs on macOS, Linux, and Windows.

---------------------------------------------------------------------------------
3. INSTALLATION INSTRUCTIONS
---------------------------------------------------------------------------------

To install Pancake, follow these steps:

1. Copy the Pancake file.
2. Add the file to a bin directory.
3. Run the `pancake` command to ensure proper installation.

---------------------------------------------------------------------------------
4. FEATURES
---------------------------------------------------------------------------------

Here are some of the key features of Pancake:

1. Pancake provides a comprehensive project management system, allowing you to view 
the list of all projects you are working on, sync multiple projects from remote repositories, 
and build and run projects on a local machine.

2. It simplifies the installation, uninstallation, and updating of development software 
and tools, eliminating the need to remember longer commands.

3. Pancake enables efficient collaboration by allowing you to share your entire 
development setup with others or back up your development environment by sharing 
the $HOME/pancake.yml file.

4. The tool offers detailed monitoring capabilities, displaying information 
about ports, process IDs (PIDs), and uptime for running projects.

---------------------------------------------------------------------------------
5. TROUBLESHOOTING
---------------------------------------------------------------------------------

If you encounter any issues, try the following steps:

1. Run `pancake help` to see the list of available commands and their usage.
2. Raise an issue on our GitHub repository: [Pancake Issues](https://github.com/a6h15hek/pancake/issues).
